一些md语法

发表于 2020-05-28

.md即markdown文件的基本常用编写语法,本blog就是用此语法来编写，因为经常忘互联网有记忆我没有所以做了个大致汇总。

基本符号

* - + >
基本上所有的markdown标记都是基于这四个符号或组合，需要注意的是，如果以基本符号开头的标记，注意基本符号后有一个用于分割标记符和内容的空格。

转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符

列表

无序列表使用 * + -作为列表标记：

* 第一项
+ 第二项
- 第三项

有序列表使用数字并加上.来表示，如：

  1. 第一项

列表嵌套

列表嵌套只需在子列表中的选项添加四个空格即可：

1.第一项
-第一项嵌套的第一个元素
-第二项嵌套的第二个元素
2.第二项
-第二项嵌套的第一个元素

引用区块

>区块引用

>>>第二层嵌套

代码块

如果是段落上的一个函数或片段的代码可以用反引号把它包起来 `，例如：
